What the Healer fixes:
✓ Missing release years in movie filenames
✓ Missing season/episode numbers in TV files
✓ Generic filenames (untitled, video, track 01)
✓ Scene group junk tokens (YIFY, RARBG, etc.)
✓ Inconsistent naming across a collection
✓ Wrong file extension casing (.MKV → .mkv)
✓ Missing resolution tags in multi-version movies
How it works:
The healer infers metadata from folder names (e.g. "Breaking Bad (2008)/Season 01/"), sibling files, and NFO companions, then applies consistent naming across the entire collection.
Short press – diagnose + heal collection
Long press (>0.5 s) – dry run (preview in log, no changes)
Dial rotate – browse diagnosed issues
Dial touch – diagnose only (no changes)
